# 常量声明

常量在声明时必须进行初始化，并且不允许对其重新赋值。

```html
<!DOCTYPE html>
<html lang="zh-Hans">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>JavaScript</title>
</head>
<body>
    <script type="text/javascript">
        const num = 233;
        console.log(num);
    </script>
</body>
</html>
```

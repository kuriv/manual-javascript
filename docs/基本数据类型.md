# 基本数据类型

JavaScript 中的基本数据类型主要包括数值类型、字符串类型、布尔类型、未定义类型、空类型。

```javascript
let num = 233;
console.log(num, typeof num);
let str = 'Hello World!';
console.log(str, typeof str);
let bool = true;
console.log(bool, typeof bool);
let undef;
console.log(undef, typeof undef);
let NULL = null;
console.log(NULL, typeof NULL);
```

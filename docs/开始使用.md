# 开始使用

执行下面的代码，输出指定的内容。

```html
<!DOCTYPE html>
<html lang="zh-Hans">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>JavaScript</title>
</head>
<body>
    <script type="text/javascript">
        document.write('Hello World!');
        alert('Hello World!');
        console.log('Hello World!');
    </script>
</body>
</html>
```

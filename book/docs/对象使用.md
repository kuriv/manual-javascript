# 对象使用

执行下面的代码，声明对象并访问其属性。

```javascript
let obj = {
    name: 'Tom',
    age: 18,
    'user-name': 'Tom'
};
console.log(obj);
console.log(obj.name);
console.log(obj.age);
console.log(obj['user-name']);
```

执行下面的代码，删除对象中的指定属性。

```javascript
let obj = {
    name: 'Tom',
    age: 18
};
console.log(obj);
delete obj.name;
console.log(obj);
console.log(obj.name);
```

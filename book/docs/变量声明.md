# 变量声明

执行下面的代码，声明变量并为其赋值。

```html
<!DOCTYPE html>
<html lang="zh-Hans">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>JavaScript</title>
</head>
<body>
    <script type="text/javascript">
        let num;
        num = 233;
        console.log(num);
    </script>
</body>
</html>
```

也可以在声明变量时直接完成初始化赋值操作。

```html
<!DOCTYPE html>
<html lang="zh-Hans">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>JavaScript</title>
</head>
<body>
    <script type="text/javascript">
        let num = 233;
        console.log(233);
    </script>
</body>
</html>
```

还可以一次性声明多个变量并为其赋值。

```html
<!DOCTYPE html>
<html lang="zh-Hans">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>JavaScript</title>
</head>
<body>
    <script type="text/javascript">
        let num = 233, str = 'Hello World!';
        console.log(num, str);
    </script>
</body>
</html>
```
